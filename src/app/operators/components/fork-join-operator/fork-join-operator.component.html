Fork join - is used to combine multiple observables into a single observable that emits an array of values, once all the observables have completed. The emitted values are in the same order as the input observables. In example below it is used to get data from two diffrent endpoints, one with data about people and second about people addresses and than it uses it to create table.

Name Age City Zip code Street {{ person.name }} {{ person.age }} {{ person.address?.city }} {{ person.address?.zipCode }} {{ person.address?.street }}
