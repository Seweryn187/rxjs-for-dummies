Catch error - is used to handle errors that occur during the execution of an observable stream. It allows you to gracefully handle errors without causing the observable to terminate and emit an error notification. When an error occurs in an observable sequence, catchError intercepts the error and allows you to handle it in some way. You can return a new observable sequence to replace the original sequence, or you can return a fallback value or default value.
